#
# IKPHP 安装说明 
#

+ 常用路径
  - 安装路径: http://yoursite/install 重新安装请删除掉 网站根目录下data/目录下的 install.lock即可重新安装
  - 前台登录: http://yoursite/index.php
  - 后台登录: http://yoursite/index.php?app=admin

  - 后台登录账号：默认是 admin@admin.com 密码：000000
+ 所有应用包都放在 Core\Apps 目录下
